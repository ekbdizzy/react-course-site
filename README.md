## React app for a website with courses.

Simple SPA on React for REST API on Django Rest Framework.

Backend:
~~~
https://github.com/ekbdizzy/course_site_api
~~~

Install dependencies:
~~~
npm install
~~~

Start project:
~~~
npm start
~~~
